# В скрипте main реализован консольный калькулятор с использованием библиотеки PyAutoGui
    - скрипт проверяет операционную систему, но фунционал реализован лишь для Windows
    - поддерживает все целые числа и операции "+" "-" "/" "*", команда "exit" для выхода
    - скрипт работает как со светлой, так и с темной темами калькулятора


## В папке проекта лежат:
    - основной скрипт main.py
    - набор png изображений кнопок для распознавания (dark_theme, light_theme)
    - файл для тестирования входных данных test.py (для автоматического тестирования без консольного ввода)


### Скрипт реализован с использованием следующих библиотек:
    - PyAutoGUI==0.9.54 — Библиотека эмулирует клики мыши, нажатия клавиш и другие взаимодействия с интерфейсом,
                          что позволяет управлять калькулятором, имитируя действия пользователя.
    - OpenCV==4.10.0 — необходима для обработки изображений, используемых для распознавания кнопок калькулятора,
                       что позволяет PyAutoGUI выполнять клики по нужным элементам.
    - Pillow==11.0.0 — используется для обработки изображений, в том числе для их
                       предварительного преобразования или изменения форматов перед использованием в PyAutoGUI.
    - pytest==8.3.4 — применяется исключительно для тестирования входных данных в скрипте test.py.
                       Она не требуется для основной работы калькулятора.
                       Изменения вводных данных производится внутри файла test.py
                       Запуск скрипта тестирования по команде pytest test.py
